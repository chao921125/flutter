# 环境搭建

```text
1、找到 Flutter 官网，https://flutter.cn/
2、点击开始使用，进入flutter的文档首页
3、选择对应的开发环境
4、获取 Flutter SDK
    在获取SDK的时候如果无法访问GitHub那么，大家直接离线下载SDK包即可，但后续更新比较麻烦，需要每一次下载包覆盖更新。
5、配置 Flutter 环境变量，这一项是必要的
    每个平台都不一样根据官网进行配置即可
6、开发环境搭建
    建议大家安装JDK、NodeJS
    开发工具选择（首推前两个）：Android Studio/Visual Studio Code/IntelliJ
    模拟器：
        Android 模拟器可以选择国产的，但建议大家选择使用Android Studio内置的避免一些不必要的麻烦
        IOS MacOS系统自带的Xcode，Windows系统下参考：https://learn.microsoft.com/zh-cn/dotnet/maui/ios/remote-simulator
7、开发工具安装 Flutter 和 Dart 插件，安装插件之后如果提示找不到 Dart 或者 Flutter，需要在开发工具里面配置
8、创建 Flutter 项目（应用）并运行到模拟器或真机
```
